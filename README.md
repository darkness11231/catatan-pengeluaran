# Catatan Pengeluaran

Aplikasi mobile untuk mencatat pengeluaran harian dengan fitur kategori, laporan, dan statistik.

## 🎯 Fitur

- ✅ Catat pengeluaran (jumlah, kategori, tanggal, catatan)
- ✅ 8 Kategori dengan icon & warna
- ✅ Database lokal SQLite (offline)
- ✅ Laporan & statistik dengan pie chart
- ✅ Filter per bulan
- ✅ Edit & hapus pengeluaran

## 🚀 Download APK

APK akan tersedia di [Releases](../../releases) setelah build selesai.

Atau download dari GitHub Actions:
1. Klik tab "Actions" di atas
2. Pilih workflow run terbaru yang berhasil (✓)
3. Scroll ke bawah, download "release-apk"

## 🛠️ Build Sendiri

```bash
# Clone repository
git clone <repo-url>
cd catatan_pengeluaran

# Install dependencies
flutter pub get

# Build APK
flutter build apk --release
```

## 💻 Teknologi

- Flutter 3.44.4
- Dart
- SQLite (sqflite)
- Provider (state management)
- fl_chart (grafik)
